
# Math, Physics & Discrete Mathematics OOP Toolkit

## Description

This project is a Java-based educational toolkit developed using Object-Oriented Programming principles. It combines important concepts from **Calculus 2, Physics 2, and Discrete Mathematics** into one structured application.

The purpose of this project is to demonstrate how complex mathematical and scientific problems can be organized, explained, and managed using software design principles.

Each problem is implemented as an individual object that contains:
- A problem title
- A short explanation of the solution steps
- The final answer

The program uses a common abstract base class and applies OOP concepts such as **inheritance and polymorphism** to create a clean and expandable structure.

## Project Goal

The main goal of this project is to combine mathematical problem-solving with programming by creating a reusable structure where new topics can easily be added in the future.

This project demonstrates how OOP can be applied to organize educational content and transform theoretical concepts into a practical software application.
